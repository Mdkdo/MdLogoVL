Documentation Logo VL Advanced

Bienvenue dans la documentation du langage Logo VL Advanced. Ce langage utilise une syntaxe Logo classique sans parenthèses pour les commandes, tout en offrant des fonctionnalités modernes.

Points clés :

  • Majuscules automatiques : L'éditeur convertit automatiquement les commandes en majuscules (ex: av 100 devient AV 100).
  • Séparateurs optionnels : Vous pouvez utiliser la virgule , pour séparer les paramètres (utile pour les nombres négatifs : FPOS 10, -20).
  • Arité intelligente : Les commandes savent combien d'arguments elles attendent (AV SIN 90 est compris comme AV(sin(90))).

Contrôle

REPETE n [ ... ]

REPETE 36 [ AV 10 TD 10 ]

SI condition [ ... ] SINON [ ... ]

SI (:x > 10) [ AV 100 ] SINON [ RE 100 ]

TANTQUE condition [ ... ]

TANTQUE (:x < 100) [ :x++ ]

CHOISIS val [ CASE v1 [ ... ] AUTRES [ ... ] ]

CHOISIS (:c) [ CASE "rouge" [ FCC "red" ] AUTRES [ FCC "black" ] ]

Variables & Procédures

Variables

Déclaration avec DONNE ou affectation directe.

DONNE :x 10 :y = 20 :x += 5

Procédures

POUR TRIANGLE :t REPETE 3 [ AV :t TD 120 ] FIN TRIANGLE 100

Récursivité

POUR ARBRE :t SI :t < 5 [ STOP ] AV :t TG 30 ARBRE :t * 0.7 TD 60 ARBRE :t * 0.7 TG 30 RE :t FIN
