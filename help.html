Structures de contrôle

Si... Sinon (if... else)

if (posx() > 100) { setcolor('red'); } else { setcolor('blue'); }

Switch

let forme = 'carré'; switch(forme) { case 'carré': polygon(4, 50); break; case 'triangle': polygon(3, 50); break; default: circle(50); }

Boucles (for, while)

Utilisez break pour sortir d'une boucle et continue pour passer à l'itération suivante.

// Boucle for for (let i = 0; i < 10; i++) { if (i === 5) continue; // Saute l'étape 5 fd(i * 10); rt(90); } // Boucle while let d = 0; while (d < 100) { fd(d); rt(45); d += 10; if (d > 80) break; // Arrête la boucle }
