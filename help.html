Documentation Logo VL Advanced

Bienvenue dans la documentation du langage Logo VL Advanced. Ce langage utilise une syntaxe Logo classique (sans parenthèses pour les commandes) tout en offrant la puissance de JavaScript pour les structures complexes.

Mouvement

Les commandes de mouvement déplacent la tortue sur le canevas.

CommandeDescription
AV nAvance de n pas. (Alias: FD)
RE nRecule de n pas. (Alias: BK)
TD degTourne à droite de deg degrés. (Alias: RT)
TG degTourne à gauche de deg degrés. (Alias: LT)
FPOS x yDéplace la tortue aux coordonnées (x, y).
FCAP degOriente la tortue vers l'angle deg.
HOMERamène la tortue au centre (0, 0).
VEVide l'écran et ramène la tortue au centre. (Alias: CS)

Structures de contrôle

REPETE (Boucle)

Répète un bloc d'instructions n fois. Accédez à l'index via _i0 (ou _i1, etc. pour les boucles imbriquées).

REPETE 4 [ AV 100 TD 90 ]

SI... SINON (Condition)

SI (:x == 10) [ AV 100 ] SINON [ RE 100 ]

TANTQUE (Boucle conditionnelle)

TANTQUE (:x < 100) [ AV :x DONNE :x (:x + 10) ]

STOP & CONTINUE

STOP permet de sortir d'une boucle immédiatement. CONTINUE passe à l'itération suivante.

Variables & Procédures

Déclaration de variables

Utilisez DONNE pour créer ou modifier une variable. Les noms de variables doivent commencer par deux points :.

DONNE :taille 100 :vitesse = 50 // Affectation directe possible

Procédures (Fonctions)

Définissez vos propres commandes avec POUR et FIN.

POUR CARRE :cote REPETE 4 [ AV :cote TD 90 ] FIN CARRE 150 // Appel de la procédure sans parenthèses

RENDS (Return)

Retourne une valeur depuis une procédure.

POUR DOUBLE :n RENDS (:n * 2) FIN

Avancé

Le langage supporte également les concepts avancés de JavaScript via une syntaxe adaptée.

  • CLASSE : CLASSE Nom [ ... ] pour définir une classe.
  • CHOISIS : CHOISIS (:val) [ CASE val1 [ ... ] AUTRES [ ... ] ] pour un switch/case.
  • FLUIDE : FLUIDE v active/désactive l'animation fluide.
CHOISIS (:couleur) [ CASE "rouge" [ FCC "red" ] CASE "bleu" [ FCC "blue" ] AUTRES [ FCC "black" ] ]
